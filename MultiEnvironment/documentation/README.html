CAFÉ: Instructions

If you just installed CAFÉ and haven't done this before, run
cafe setup
from the command line.

This will generate a new folder called "setup" with instructions to add right click shortcuts and notepad features that come with CAFÉ

Here are some other commands you may find useful: cafe run #Runs your entire project. Must be executed from the root directory of the project (the directory that has the features folder in it)
cafe smoketest #Only runs tests in the SmokeTest directory
cafe run path/to/feature/file #Only runs the specified feature file
cafe run path/to/feature/file:line_number #Only runs the scenario that starts at the provided line number
